Azure Configuration Required

The Azure AI chatbot needs to be configured before use. Please set up your Azure AI credentials to continue.

Configuration Steps:

  1. Go to the Settings page
  2. Get your Azure AI Foundry endpoint and API key from Azure portal
  3. Enter your Azure AI Foundry endpoint URL
  4. Enter your API key
  5. Configure your preferred model settings
  6. Save configuration
Configure Settings

Azure AI Foundry Endpoint Format:

https://your-endpoint.cognitiveservices.azure.com/
